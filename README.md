# PyConZA documentation

This documentation site was created using [Mkdocs](https://www.mkdocs.org/).

## Running locally

Installation 

```
uv sync
```

Then to serve mkdocs locally:

```
uv run mkdocs serve
```

Or if you would prefer to activate the virtual environment:

```
source .venv/bin/activate 
mkdocs serve
```

The site will then serve at [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
